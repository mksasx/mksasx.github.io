Hugo建站指南#

文章模板#

---
title: "{{ replace .Name "-" " " | title }}"
date: {{ .Date }}
lastmod: {{ .Date }}
author: ["Sulv"]
categories:
- 分类1
- 分类2
  tags:
- 标签1
- 标签2
  description: ""
  weight: # 输入1可以顶置文章，用来给文章展示排序，不填就默认按时间排序
  slug: ""
  draft: false # 是否为草稿
  comments: true
  showToc: true # 显示目录
  TocOpen: true # 自动展开目录
  hidemeta: false # 是否隐藏文章的元信息，如发布日期、作者等
  disableShare: true # 底部不显示分享栏
  showbreadcrumbs: true #顶部显示当前路径
  cover:
  image: ""
  caption: ""
  alt: ""
  relative: false
---
